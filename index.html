GPS Drag – Website

GPS Drag is a game made for GPS Devices. It's porting app into a fake map, so the device installs it. Gameplay instructions are built in game.

In very near future I will add leaderboards, an C++ game file (to download button), hardest levels tab, Music GPSizer (normal music have got too big bitrate for GPS, so this get it lower), level downloader (Yes, you can build and submit levels on PC and Smartphones!)

It's an Auto-Side-Scrolling-Rhtyhm-Runner.

Do you want to download some levels for your GPS? If yes, then check this out.
